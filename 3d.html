USB Port

Ports & Connectors

Universal Serial Bus (USB) is a standard connection interface for data transfer and power supply. USB ports enable plug-and-play connectivity for various peripherals.

USB Connector

Ports & Connectors

A USB connector is the end part of a USB cable. It is used to connect two devices for data transfer and charging. USB connectors carry data signals and electric power. Common USB connectors include USB-A, USB-B, USB-C, Micro-USB, and Mini-USB. USB-C is the latest connector and can be plugged in either direction.

C-type Port

Ports & Connectors

A USB Type-C port is the socket found on devices like mobiles, laptops, and tablets.
It supports fast data transfer, fast charging, and video output.
The Type-C port is small and reversible (no up or down side).
It is used in modern devices.

C-type connector

Ports & Connectors

A USB Type-C connector is the plug at the end of a USB-C cable.
It fits into the Type-C port from any direction.
It can carry power, data, audio, and video signals.
One Type-C connector can be used for charging and data transfer.

Ethernet Port

Ports & Connectors

An Ethernet port is a socket used to connect a device to a wired network.
It provides a stable and fast internet connection.
Ethernet ports are commonly found on computers, routers, and switches.
It uses a LAN (Local Area Network) cable.

Ethernet Connector

Ports & Connectors

An Ethernet connector is the plug at the end of an Ethernet cable.
It is also called an RJ-45 connector.
The Ethernet connector connects the cable to the Ethernet port.
It helps in data communication between network devices.

VGA Port and Connector

Ports & Connectors

VGA (Video Graphics Array) is used to connect a computer to a monitor or projector.
A VGA port is the socket on the computer or display device.
A VGA connector is the plug at the end of the VGA cable.
It uses a 15-pin (DE-15) connector.
VGA carries analog video signals only (no audio).
It is an older display interface, mostly replaced by HDMI and DisplayPort.

Power Port & Connector

Ports & Connectors

A power cable is used to supply electrical power to a device.
A power port is the socket on the device where power is received.
A power connector is the plug at the end of the power cable.
It connects the device to a power source like a wall socket or adapter.
Power cables are used in computers, monitors, printers, and chargers.
The main function is to provide safe and continuous electricity to the device.

PS/ 2 Connector

Ports & Connectors

PS/2 is a type of connector used for keyboard and mouse.
A PS/2 port is the round socket on a computer.
A PS/2 connector is the round plug at the end of the PS/2 cable.
It has 6 pins for data communication.
Usually, purple is for keyboard and green is for mouse.
PS/2 is an older interface, now mostly replaced by USB.

Serial communication port & Connector

Ports & Connectors

Serial communication is a method of sending data one bit at a time.
A serial port is a socket used for serial data communication.
A serial connector is the plug attached to the serial cable.
Common serial connectors are DB-9 (9-pin) and DB-25 (25-pin).
Serial ports were used for modems, mice, and industrial devices.
It is an older communication method, now mostly replaced by USB.

Parallel communication port & Connector

Ports & Connectors

Parallel communication sends multiple bits of data at the same time.
A parallel port is a socket used for parallel data transfer.
A parallel connector is the plug at the end of the parallel cable.
It commonly uses a DB-25 (25-pin) connector.
Parallel ports were mainly used for printers and scanners.
It is an older technology, now mostly replaced by USB.

Processor

Computer Parts

A processor is the brain of the computer.
It is also called the CPU (Central Processing Unit).
The processor executes instructions and performs calculations.
It controls all the operations of the computer.
The speed of a processor is measured in GHz.
Common processor brands are Intel and AMD.

RAM Module

Computer Parts

Random Access Memory (RAM) is volatile memory used for temporary data storage. It provides fast read and write access to the processor, directly impacting system performance.

Hard Disk

Computer Parts

A hard disk is a secondary storage device.
It is used to store data permanently.
It stores the operating system, software, and files.
Hard disks have large storage capacity.
Data remains saved even when the power is off.
HDD is slower than SSD, but cheaper.

SSD

Computer Parts

An SSD is a storage device used to store data permanently.
It is faster than a hard disk (HDD).
SSD has no moving parts, so it is more durable.
It provides quick boot and fast file access.
SSD consumes less power.
Common types are SATA SSD and NVMe SSD.

GPU(RTX 3060)

Computer Parts

RTX 3060 is a graphics processing unit (GPU) made by NVIDIA. It is used for gaming, video editing, and 3D rendering.
It supports ray tracing and DLSS technology.
RTX 3060 has 12 GB GDDR6 VRAM.
It delivers high-performance graphics for modern games.
It is suitable for 1080p and 1440p gaming.

SMPS

Computer Parts

SMPS stands for Switched Mode Power Supply. It is used to convert AC power into DC power.
SMPS supplies power to all computer components.
It is efficient and compact in size.
SMPS protects the system from overvoltage and short circuits.
It is commonly used in desktop computers.

Computer cabinate

Computer Parts

A computer cabinet is the outer box of a desktop computer. It holds and protects all internal components.
Main components inside are motherboard, CPU, RAM, HDD/SSD, SMPS, and GPU.
It helps in proper airflow and cooling.
Computer cabinets come in different sizes like ATX, Micro-ATX, and Mini-ITX.
It is also called a computer case or system unit.
