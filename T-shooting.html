Problem #1

Computer Not Turning ON

When your computer doesn't power on at all, it's usually a power supply or hardware connection issue [web:1][web:3].

Step-by-Step Solution:

  • Check the power source: Verify that the wall outlet is working by testing it with another device like a lamp or monitor [web:3].
  • Inspect power cables: Ensure the power cable is securely connected to both the computer and the outlet. Check if the power strip is turned on [web:3].
  • Discharge residual power: Unplug the power cable, hold the power button for 15-20 seconds to drain any remaining power, then reconnect and try again [web:3].
  • Check PSU connections: Open the case and verify all internal power supply cables are firmly connected to the motherboard and components [web:5].
  • Test the power supply: If available, use the PSU's Built-In Self-Test (BIST) button or try a different power supply to rule out PSU failure [web:3][web:5].
  • Clear CMOS: Reset the BIOS by removing the CMOS battery for a few minutes, then reinstalling it [web:5].
Problem #2

No Display on Monitor

Your computer powers on but the monitor shows nothing. This is often caused by cable issues or graphics card problems [web:6][web:9].

Step-by-Step Solution:

  • Check monitor power: Ensure the monitor is turned on and its power cable is securely connected [web:9].
  • Verify cable connections: Check that HDMI, DisplayPort, or VGA cables are connected to the graphics card (not the motherboard) and monitor [web:6][web:9].
  • Try different cables and ports: Test with a different video cable or use another port on your graphics card [web:9].
  • Disconnect peripherals: Remove all USB devices, external drives, and other peripherals, then restart the computer [web:6].
  • Reseat the graphics card: Turn off the PC, remove the graphics card, clean the contacts, and firmly reinstall it ensuring it clicks into place [web:8][web:9].
  • Reseat RAM modules: Remove and reinstall RAM sticks one at a time to test if faulty memory is causing the issue [web:6][web:9].
  • Reset CMOS battery: Remove the coin-cell battery from the motherboard for 10-20 seconds to reset BIOS settings [web:6].
Problem #3

Keyboard or Mouse Not Working

When your keyboard or mouse stops responding, it could be a connectivity, driver, or power issue [web:7][web:10].

Step-by-Step Solution:

  • Restart your computer: Use keyboard shortcuts (Windows + X, then navigate to Restart) or force restart by holding the power button [web:7].
  • Check connections: For wired devices, ensure cables are securely plugged in. For wireless, check battery levels and Bluetooth/transmitter connections [web:7].
  • Enable Bluetooth: Press Windows + A to open Action Center and turn on Bluetooth if using wireless devices [web:7].
  • Try different USB ports: Plug the keyboard/mouse into different USB ports on your computer [web:7].
  • Update drivers: Press Windows + X, select Device Manager, expand "Keyboards" and "Mice and other pointing devices," right-click your device, and select "Update Driver" [web:7].
  • Reinstall drivers: In Device Manager, right-click the device, select "Uninstall device," restart your computer, and Windows will automatically reinstall the driver [web:7].
  • Test with another device: Try a different keyboard or mouse to determine if the issue is hardware or software-related [web:7].
Problem #4

USB Device Not Detected

When Windows doesn't recognize your USB device, it's typically a driver, port, or power setting issue [web:11][web:14].

Step-by-Step Solution:

  • Restart your computer: A simple reboot can often resolve temporary USB recognition issues [web:15].
  • Try different USB ports: Test the device in multiple USB ports, preferably on the back of your computer for desktop PCs [web:15].
  • Check the device on another computer: Verify if the USB device is faulty by testing it on a different machine [web:15].
  • Update USB drivers: Open Device Manager, expand "Universal Serial Bus controllers," right-click each USB Root Hub, and select "Update driver" [web:14].
  • Disable USB selective suspend: Go to Control Panel > Power Options > Change plan settings > Change advanced power settings > USB settings > USB selective suspend setting, and set to "Disabled" [web:11][web:15].
  • Change drive letter: Open Disk Management, right-click the USB drive, select "Change Drive Letter and Paths," and assign a new letter [web:11][web:14].
  • Run Device Diagnostic: Press Windows + R, type "msdt.exe -ID DeviceDiagnostic," and follow the troubleshooter [web:13].
  • Reinstall USB controllers: In Device Manager, uninstall all USB Root Hubs and restart your computer for automatic reinstallation [web:11].
Problem #5

Computer Running Very Slow

Slow performance is usually caused by too many programs running, insufficient memory, or accumulated system junk [web:16][web:19].

Step-by-Step Solution:

  • Restart your computer: Close all programs and restart to clear memory and terminate unnecessary background processes [web:16].
  • Close unnecessary programs: Press Ctrl + Shift + Esc to open Task Manager, identify programs using high CPU/memory, and end those tasks [web:19].
  • Disable startup programs: In Task Manager, go to the Startup tab and disable programs that don't need to run at boot [web:19].
  • Clear temporary files: Press Windows + R, type "temp," delete all files, then repeat with "%temp%" [web:20].
  • Run Disk Cleanup: Search for "Disk Cleanup," select your system drive, check all boxes, and delete temporary files [web:16].
  • Defragment hard drive: For traditional HDDs (not SSDs), use the built-in defragmentation tool to optimize disk performance [web:19].
  • Check for malware: Run a full system scan with Windows Defender or your antivirus software [web:19].
  • Update Windows: Install all available Windows updates which may include performance improvements [web:14].
  • Upgrade RAM: If your computer still runs slow after these steps, consider adding more RAM for better multitasking [web:19].
Problem #6

Virus or Malware Issue

Malware infections can cause various problems including slow performance, pop-ups, and data theft [web:17][web:20].

Step-by-Step Solution:

  • Boot into Safe Mode: Restart your computer and press F8 before the Windows logo appears, then select Safe Mode [web:17].
  • Clear temporary files: Press Windows + R, type "temp," delete all files, then type "%temp%" and delete those files too [web:20].
  • Install/Update antivirus: Download and install a reputable antivirus like Windows Defender, Malwarebytes, or Microsoft Security Essentials [web:17].
  • Run full system scan: Perform a complete scan of your system and remove all detected threats [web:17][web:20].
  • Use Malware removal tools: Download and run specialized tools like ComboFix, TDSSKiller, or Malwarebytes to catch stubborn malware [web:17].
  • Uninstall suspicious programs: Use Revo Uninstaller to remove malicious programs and their remnants completely [web:17].
  • Run System File Checker: Open Command Prompt as admin and run "sfc /scannow" to repair corrupted system files [web:15].
  • Update all software: Ensure Windows, browsers, and all applications are updated to patch security vulnerabilities [web:17].
  • Change passwords: After cleaning, change all important passwords as they may have been compromised [web:20].
Problem #7

No Internet Connection

Internet connectivity issues can stem from router problems, network settings, or ISP issues [web:21][web:24].

Step-by-Step Solution:

  • Check other devices: Test if other devices can connect to verify if the issue is with your computer or network [web:24].
  • Restart your modem and router: Unplug both devices, wait 30 seconds, plug in the modem first, wait 60 seconds, then plug in the router [web:21][web:24].
  • Check cable connections: Ensure Ethernet cables are securely connected and not damaged. For fiber, check the SPDIF module; for cable, tighten the coaxial connection [web:21].
  • Forget and reconnect to Wi-Fi: Go to Network Settings, forget your Wi-Fi network, then reconnect by entering the password again [web:22][web:25].
  • Run Network Troubleshooter: Right-click the network icon in the system tray and select "Troubleshoot problems" [web:22].
  • Update network drivers: Open Device Manager, expand "Network adapters," right-click your adapter, and select "Update driver" [web:22].
  • Reset network settings: Go to Settings > Network & Internet > Status > Network reset, then restart your computer [web:22].
  • Disable VPN/Firewall temporarily: Turn off VPN and firewall temporarily to see if they're blocking the connection [web:24].
  • Contact your ISP: If nothing works, contact your Internet Service Provider to check for outages or account issues [web:21][web:24].
Problem #8

Blue Screen of Death (BSOD)

The Blue Screen of Death indicates critical system errors caused by hardware failures, driver issues, or corrupted files [web:26][web:29].

Step-by-Step Solution:

  • Note the error code: Write down the STOP code displayed on the blue screen for troubleshooting reference [web:26].
  • Boot into Safe Mode: Restart and press F8 repeatedly before Windows loads, then select Safe Mode [web:26].
  • Run System File Checker: Open Command Prompt as admin and run "sfc /scannow" to repair corrupted system files [web:29].
  • Run DISM tool: In Command Prompt (admin), run "DISM /Online /Cleanup-Image /RestoreHealth" to repair the Windows image [web:29].
  • Check disk for errors: Run "chkdsk C: /f" in Command Prompt to scan and fix disk errors (replace C: with your system drive) [web:26].
  • Update drivers: In Device Manager, update all drivers, especially graphics card, chipset, and storage drivers [web:26].
  • Uninstall recent updates: If BSOD started after a Windows update, uninstall the recent update from Settings > Update & Security > View update history [web:26].
  • Check hardware: Reseat RAM modules and graphics card to ensure proper connection [web:8].
  • Restore Windows: Use System Restore to revert to a point before the BSOD started occurring [web:26].
Problem #9

BIOS Error

BIOS errors prevent your computer from booting properly and can be caused by corrupted firmware, hardware changes, or failed updates [web:27][web:30].

Step-by-Step Solution:

  • Reset CMOS: Turn off the computer, unplug it, remove the CMOS battery from the motherboard for 20 minutes, then reinstall it [web:27].
  • Check hardware connections: Ensure all cables, RAM, and expansion cards are properly seated in their slots [web:27].
  • Remove graphics card: Try booting without the dedicated graphics card to see if it's causing conflicts [web:27].
  • Replace CMOS battery: If the battery is old (3-5 years), replace it with a new CR2032 battery [web:27].
  • Update BIOS: Download the latest BIOS update from your motherboard manufacturer's website onto a USB drive [web:27].
  • Flash BIOS: Boot into BIOS (usually by pressing Del or F2), navigate to the BIOS update utility, and flash the BIOS using the USB drive [web:27].
  • Use backup BIOS: If your motherboard has a backup BIOS chip, switch to it using the motherboard's jumper or button [web:27].
  • Replace BIOS chip: As a last resort, you may need to replace the BIOS chip or the entire motherboard if corruption is severe [web:27][web:30].
Problem #10

File Missing or Corrupted

System files can become corrupted due to malware, improper shutdowns, or disk errors, preventing Windows from starting properly [web:26][web:29].

Step-by-Step Solution:

  • Boot into Safe Mode: Restart your computer and press F8 before Windows loads, then select Safe Mode [web:26].
  • Run System File Checker: Open Command Prompt as administrator and run "sfc /scannow" to scan and repair corrupted system files [web:29].
  • Run DISM tool: Execute "DISM /Online /Cleanup-Image /RestoreHealth" in Command Prompt (admin) to repair the Windows component store [web:29].
  • Check disk for errors: Run "chkdsk C: /f /r" to scan for bad sectors and fix file system errors (replace C: with your drive letter) [web:26].
  • Use Windows Recovery: Boot from a Windows installation USB, select "Repair your computer," then choose "Startup Repair" [web:26].
  • Restore from backup: If you have system backups, restore your system to a point before the file corruption occurred [web:26].
  • Reinstall the application: If a specific program file is corrupted, uninstall and reinstall that application [web:20].
  • Perform system restore: Use System Restore to revert Windows to a previous working state [web:14].
  • Clean install Windows: As a last resort, backup your data and perform a clean Windows installation [web:26].
